Podcast Generator

Übergib eine URL und erzeuge daraus einen 2-minütigen Podcast mit zwei Speakern – direkt im Browser mit Azure KI.

Konfiguration

Hinterlege deine Azure-Zugangsdaten. Die Werte werden ausschließlich lokal im Browser gespeichert.

Azure OpenAI

Azure Speech

Stimmen & Proxy

Hinweise

  • API-Keys werden nur lokal im Browser-LocalStorage gespeichert – nicht auf einem Server.
  • Viele Websites blockieren direkte Browser-Anfragen (CORS). Nutze den Proxy-Platzhalter {url}.
  • Empfohlene Region für Azure Speech: westeurope.

Podcast erzeugen

Gib eine Website-URL an. Der Inhalt wird geladen, per Azure OpenAI als SSML-Podcast aufbereitet und anschließend mit Azure Speech synthetisiert.

Quelle

Status

Bereit.

Generiertes SSML

Audio

Ablauf

  1. Website-Inhalt laden (direkt oder über Proxy)
  2. Azure OpenAI erzeugt valides SSML mit zwei Stimmen (~2 min)
  3. Azure Speech synthetisiert das Audio-MP3
  4. Player und SSML werden direkt angezeigt

Logs & Debugging

Alle Anfragen und Antworten zu Azure OpenAI und Azure Speech werden hier protokolliert. API-Keys werden nicht geloggt.
